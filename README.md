# Spring Boot WireMock OAuth2 Mocking Demo

Requires a JVM of version 17 or above to be installed.

## How to run
Start the OAuth2 mock:

```bash
./gradlew runMocks
```

Start the app:

```bash
./gradlew bootRun
```

Open [http://localhost:8080](http://localhost:8080/) in your browser to start the demo.
